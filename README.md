# torbjorn-ai

Personlig webbplats som dokumenterar hur jag arbetar praktiskt med AI inom marknad, growth, data, automation och företagsbyggande.

## Quick Start

```bash
npm install
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000)

## Keystatic Editor

För att redigera innehåll visuellt, gå till [http://localhost:3000/keystatic](http://localhost:3000/keystatic)

## Struktur

- **Start** (`/`) – Introduktion och översikt
- **Arbete** (`/arbete`) – Dokumenterade processer
- **Riktning** (`/riktning`) – Min syn på AI och arbete

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Keystatic (content management)

## Deployment

Push till `main` → Vercel auto-deploy

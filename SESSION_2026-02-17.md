## Rendering-fix (LÖST)

**Problemet:** Brödtext var nästan osynlig (extremt ljusgrå) mot vit bakgrund.

**Orsak:** I `globals.css` hade `p`-taggen bara `text-base` (font-size) men ingen färg.

**Lösning:**
```css
p {
  @apply text-base leading-relaxed text-text;  /* Lade till text-text */
}

.prose {
  @apply max-w-content text-text;  /* Lade till text-text */
}
```

## Stilguide för innehåll (från session 1)

### Gör:
- Löptext dominerar (60%) med punktlistor som komplement
- Korta stycken (2-4 meningar) för mobilläsning
- Mänsklig, konversationell ton
- Konkreta exempel istället för abstrakta begrepp
- Orsak-verkan-kedja för tekniska förklaringar

### Undvik:
- Punktform som huvudsakligt innehåll
- Saxat innehåll från tekniska tabeller
- För mycket teknisk detalj (container-ID:n etc.)
- Långa, komplexa stycken
